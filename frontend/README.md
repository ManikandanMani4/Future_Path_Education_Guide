## 📸 Application Flow (Step-by-Step)

This application guides the user from authentication to AI-powered roadmap generation in a seamless flow.

---

### 🔐 Step 1: User Login
The user securely logs into the system to begin their personalized journey.

![Login](assets/login.png)

---

### 📝 Step 2: Enter Details & Choose Ambition
Users provide their personal details and select their career goal along with the time frame.

![Form](assets/form.png)

---

### 🤖 Step 3: AI Generates Personalized Roadmap
The system uses AI to generate a structured, detailed roadmap tailored to the user's ambition.

---

### 📊 Step 4: Interactive Dashboard
The roadmap is displayed in an interactive format:
- 📅 Year-wise breakdown
- 🔽 Expandable learning phases
- 📚 Topics and 🛠️ project suggestions

![Dashboard](assets/dashboard.png)
